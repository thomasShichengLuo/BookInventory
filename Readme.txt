**Scenario:**

Please build a web application to manage a library's book inventory and user borrowing system. The application should allow users to perform the following operations:

- **Manage Books**: Create, Read, Update, Delete book entries.

- **User Management**: Register, login, and view profile.

- **Borrowing System**: Users can borrow and return books.


This solution build on ASP.NET Core, .NET 8, EF Core 8, Blazor and Identity, including authentication and authorization.
It handles of validation, error scenarios, and edge cases. Unit test added.
It uses a MS SQL database with code first.
